## Overview

A repository of docker files.

<br />

## Usage

### Build

```
docker build -t ${DOCKER_USERNAME}/${IMAGE_NAME}:${TAG} ${DIRECTORY}
```

### Push

```
docker push ${DOCKER_USERNAME}/${IMAGE_NAME}:${TAG}
```
